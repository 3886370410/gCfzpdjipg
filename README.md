# 项目简介
基于python的二手房推荐可视化系统赠ppt+开题+任务书

二手房推荐系统 大数据项目 计算机项目 
无爬虫
Python项目 flask项目 vue项目

开发语言：Python

数据库：mysql

前端框架：vue

后端框架：Flask
系统核心功能（完整功能可以看展示视频）：

依据管理员的权限需求以及对系统的管理需求进行分析，主要包括了个人中心、用户管理、二手房信息管理、系统管理等。

用户的需求进行分析，包括首页、二手房信息、系统公告、个人中心等。

## 核心功能
- 用户管理
- 二手房信息管理
- 个人中心
- 系统公告

## 适用场景
- 二手房市场数据分析
- 二手房信息展示与查询
- 用户个人资料管理与互动

## 技术特点
- 开发语言：Python
- 后端框架：Flask
- 前端框架：Vue
- 数据库：MySQL
- 系统具备可视化功能，便于用户直观理解数据
- 支持远程部署，需额外收费
